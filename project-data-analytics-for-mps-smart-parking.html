Data Analytics for Mornington Peninsula Shire Smart Parking Enforcement

To meet the challenges of increasing demand on parking and effectively managing parking in towns, the Mornington Peninsula Shire (MPS) is developing a smart parking system which can integrate parking sensor data for parking managment and enforcement. This project consists of the following work packages helping MPS understand current enforcement situations and discover overstay patterns for evidence-based enforcement decision-making.

  • PIN (Parking Infringement Notice) Revenue Baseline Modelling: Parking enforcement data from Pinforce contains the detailed information about parking infringement notices issued over the past years across the Mornington Peninsula Shire. We use the parking enforcement data to build a revenue baseline to help MPS understand current situations and enables the assessment of new initiatives.
  • Overstay Hotspots Modelling: Smart parking sensor data contains the information about parking arrival times and durations for each bay with a sensor installed. We use sensor data APIs to collect smart parking sensor data and conduct data analytics to identify time-dependent hotspots of overstay for supporting enforcement decision making.
  • Parking Enforcement Dashboard: A dashboard is developed to visualize the outcomes of the above two work packages. The dashboard will provide interfaces for user interactions, for example, viewing PIN revenue breakdowns and querying overstay hotspots. The dashboard has the potential to be further upgraded to an enforcement revenue forecasting system.
